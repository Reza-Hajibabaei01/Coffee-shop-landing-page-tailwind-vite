# Landing Page برای سایت کافی‌شاپ

این پروژه یک صفحه فرود (Landing Page) برای سایت کافی‌شاپ است که با استفاده از ابزارهای Vite و Tailwind CSS طراحی شده است. این صفحه به صورت ریسپانسیو است و به خوبی در مانیتورهای بزرگ و همچنین حالت موبایل نمایش داده می‌شود.

## ویژگی‌ها

- طراحی ریسپانسیو برای نمایش صحیح در دستگاه‌های مختلف (مانیتور بزرگ و موبایل)
- استفاده از Vite برای سرعت بارگذاری و توسعه سریع
- استفاده از Tailwind CSS برای طراحی سریع و مقیاس‌پذیر
- افزودن انیمیشن‌ها با استفاده از AOS (Animate On Scroll)
- استفاده از React برای پیاده‌سازی ساختار کامپوننتی
- استفاده از React Slick برای ایجاد اسلایدرهای زیبا

## نصب پروژه
